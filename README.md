# ⏳ Minimalist CSS Loader (Componente de Carregamento)

> Um componente de interface focado em User Experience (UX), apresentando uma animação de carregamento fluida e moderna construída puramente com HTML5 e CSS3.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo principal foi desenvolver um **Loader** (indicador de carregamento) que seja visualmente atraente e extremamente leve.

O foco técnico aqui é a performance: ao utilizar apenas CSS para a animação, o componente não sobrecarrega a thread principal do navegador, garantindo uma transição suave mesmo em conexões mais lentas ou dispositivos com hardware limitado.

## ✨ Destaques e Aprendizados Técnicos

* **Arquitetura Minimalista:** Implementação baseada em uma única tag HTML (`.loader`), demonstrando a capacidade de criar elementos complexos com marcação mínima e eficiente.
* **Animações de Keyframes:** Uso avançado de `@keyframes` para controlar ciclos de animação infinitos, rotações e transições de estado visual.
* **Micro-interações de Feedback:** Foco na psicologia do usuário, fornecendo um indicador visual de que o sistema está processando informações, o que reduz a taxa de rejeição.
* **Código Limpo e Escalável:** Estrutura organizada seguindo padrões profissionais, com a semântica no `index.html` e a lógica visual isolada no diretório `css/style.css`.
* **Design Vetorial:** Por ser construído com CSS, o componente é totalmente escalável e mantém a nitidez em qualquer resolução de tela ou densidade de pixels.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação minimalista.
* **CSS3:** Animações, transformações geométricas, posicionamento e estilização avançada.
